==================
   Tic Tac Toe
==================

Made By: Israel Felhandler

This project is a terminal driven Tic Tac Toe Game that can be played with 0, 1, or 2 human players, with A.I as the possible oponent.
In the terminal, write [-h -h] for human vs human, [-c -c] for cpu vs cpu, and any combination of the two.
